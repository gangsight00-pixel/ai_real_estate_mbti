# 부동산 투자성향 MBTI

매매·경매 투자에 활용하는 **부동산 투자성향 진단 테스트**. 몇 가지 질문에 답하면 나의 투자성향 유형이 나오고, 유형별 강점·약점·맹점과 실전 매매/경매 전략, 그리고 경매 물건 검색조건까지 제안합니다. **본인 투자 판단용** 도구입니다.

> 🔗 **온라인 테스트**: https://earthskyisbig.github.io/auction_mbti/
> 💻 **로컬 실행**: `부동산투자성향테스트.html`(= `index.html`)을 브라우저로 열면 됩니다. 설치 불필요, 단일 파일.

## 특징

- **매매 / 경매 트랙 분리** — 두 투자 방식의 성향을 각각 진단 (각 16유형, 총 32유형)
- **채점 엔진** — 축별 점수 합산으로 유형 판별, 동점 경계형 표시
- **유형 프로필 + 활용 가이드** — 강점·약점·**맹점**, 적합 물건, 함정, 매수/입찰 전 체크리스트
- **부동산 규제 반영** — 2025 6·27 / 9·7 / 10·15 대책 + 2026-06-30 규제지역 추가지정(화성 동탄구·용인 기흥구·구리시)을 근거로, 보유주택수·규제지역에 따른 대출제한·세금중과·실거주의무 경고
- **경매 검색조건 생성** — 진단 결과 + 입력값(자금·지역·규제·보유상황)을 법원경매 검색 필터(지역·가격대·물건종류·유찰횟수·권리난이도)로 변환, JSON 내보내기

## 유형 축

**매매 트랙** (코드 예: `M-SHRG`)
| 축 | 두 극 |
|---|---|
| 리스크 | 안정 `S` ↔ 공격 `A` |
| 기간 | 장기보유 `H` ↔ 단기회전 `T` |
| 목적 | 실거주겸용 `R` ↔ 순수투자 `P` |
| 자금 | 레버리지 `G` ↔ 자기자본 `E` |

**경매 트랙** (코드 예: `A-scnr`) — 투자자금·투자용도·투자지역·주택보유상황·경매경험 반영
| 축 | 두 극 |
|---|---|
| 자금 | 소액분산 `s` ↔ 대액집중 `b` |
| 난이도 | 안전입찰 `c` ↔ 특수물건 `x` |
| 지역 | 연고지 `n` ↔ 전국추적 `w` |
| 용도 | 실수요·보유 `r` ↔ 단기매각 `f` |

## 프로젝트 구조

```
부동산투자성향테스트.html   최종 웹 테스트 (index.html과 동일)
_workspace/                  중간 산출물 (유형 체계·문항/채점·프로필·규제·검색기준·QA)
.claude/                     에이전트 팀 하네스 (agents 6 + skills 6 + 오케스트레이터)
CLAUDE.md                    하네스 포인터 + 변경 이력
```

에이전트 팀(유형 아키텍트 → 진단 설계 → 콘텐츠 → 검색기준 매퍼 → 웹 빌더 → QA)이 협업해 설계·제작하며, 각 산출물의 경계면 정합성(유형코드↔문항↔프로필↔웹채점)을 QA가 교차검증합니다.
